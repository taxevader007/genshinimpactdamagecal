Olimpiadas de mejor daño provocado en Genshin Impact 

El objetivo de este programa es desarrollar una calculadora de daño que le permita poder experimentar como podría alcanzar el daño más alto en el juego que son: transformativas, amplificativas y aditivas


Implementar las fórmulas de daño para cada una de las reacciones teniendo en cuenta los calculos de cada formula del daño 

Tambien se necesita calcular el daño realizado por un personaje o por un equipo

calcular según un tipo de reacción según el multiplicador de resistencia de un enemigo.

guardar los valores de los últimos 10 daños registrados en el sistema para su posterior consulta (sistema de memoria simple) 


Mostrar el daño más alto realizado en la calculadora hasta el momento. 

variables
-multiplicador de resistencia (Este último debe ser generado de manera aleatoria en un rango entre 0.5 y 2 únicamente.)

input
-cuantos personajes hay 1-4
-nombre del personaje 
-ataque del personaje
-nivel del personaje
-PDC
-DC
-ME

output

-Daño base
-Daño Transformativo 
-Daño Amplificador
-Daño Aditivo

Danobase = AtaquePersonaje * (1 + PDC + DC) output

Además, para las reacciones transformativas el daño también se ve influenciado por el multiplicador de resistencia de los enemigos. 

Transformativas: 

Danobase = AtaquePersonaje * (1 + PDC + DC)

()Daño Transformativo = Multiplicador Transformativo*Nivel personaje *(1 + %BonoME) *Multiplicador resistencia 

%BonoME = 16 * Maestría Elemental / (Maestría Elemental + 2000) 

Los Multiplicadores Transformativos son los siguientes:  

0.25 para quemadura 

0.50 para superconductor 

0.6 para torbellino 

1.2 para electro cargado 

1.5 para cristalización 

2.0 para sobrecargado

2.0 para florecimiento 

3.0 para sobre florecimiento y super quemadura. 

Amplificativas  

Los Multiplicadores de Reacción de Amplificación son los siguientes: 
1.5 para vaporización 
2.0 para derretir. 

Multiplicador de amplificación = Multiplicador de reacción de Amplificación *(1 + %BonoME )  
 
%BonoME = 2.78* Maestría Elemental / (Maestría Elemental + 1400) 

Finalmente 

()Daño Amplificador = Daño Base*Multiplicador de reacción Amplificación. 

Multiplicadores de Reacción de Amplificación 

()Daño Aditivo = Multiplicador Aditivo* Nivel Personaje * (1 + %BonoME)*Multiplicador resistencia 

Los Multiplicadores Aditivos son los siguientes: 
1.15 para intensificación
1.25 para propagación

%BonoME = 5 * Maestría Elemental/ (Maestría Elemental + 1200)
